This will fetch the todo list for the employee with ID 1.

Script Description
The script takes the employee ID as a command-line argument and uses it to fetch the employee's details and todo list from the JSONPlaceholder API.
It calculates the number of completed tasks by iterating through the todo list and counting the tasks with a completed status.
Finally, it prints the employee's name, the number of completed tasks, and the titles of the completed tasks.
Dependencies
Python 3
requests (can be installed with pip install requests)
JSONPlaceholder API
The script uses the JSONPlaceholder API to retrieve employee details and todo lists.
